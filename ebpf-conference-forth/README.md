## 大会介绍

各位 eBPF 技术爱好者：

eBPF 作为 Linux 内核侧的革命性技术，已经在可观测性、网络管理、安全防护和系统性能优化等领域取得了成功并体现了巨大潜力。为促进 eBPF 技术在我国的研究与应用，我们秉承自由、开放、共享与创新的理念，成功举办了 3 届 “eBPF 开发者大会”，累计汇聚了来自高校、企业和开源社区的专家学者分享约 90 余场精彩报告，吸引数万技术爱好者线上线下共同参与。

2026 年，我们将迎来 “第四届 eBPF 开发者大会”！会议主题的主基调是：**eBPF + AI，重塑系统性能与安全新生态**。大会将以更丰富多元的议题设置、更广泛深入的跨界参与、更具深度的技术研讨，致力打造一场国内 eBPF 技术交流与应用推广盛会。

本次会议遵循开源社区自由、开放、共享的理念，不收会议费；住宿统一安排，费用自理。参会报名链接和微信群见下方二维码。
